# Travel_And_Tourism
Travel and Tourism Management System – Java Web Application

A full-stack Travel and Tourism Management System developed for **Pawan Travels and Tourism**, Uttur, Kolhapur as part of the B.Sc. Computer Science Final Year Project at Shivaji University, Kolhapur (2024–2025).

---

## 📌 About The Project

Developed a Travel and Tourism Management System for Pawan Travels and Tourism, Uttur, Kolhapur. The application digitizes the existing manual booking and record-keeping process, enabling customers to browse travel packages, make bookings, and process payments online.

## ✨ Features

### 👤 User Module
- User Registration & Login
- View Travel Packages
- View Routes
- Book a Package
- Online Payment (UPI & Credit/Debit Card)
- Cancel Booking
- Enquiry Form
- Feedback Form
- Booking List (Update / Delete)

### 🔐 Admin Module
- Admin Registration & Login
- Add / Update / Delete Packages
- Add / Update / Delete Routes
- View Booking List
- View Enquiry List
- View Feedback List
- Generate Reports (Booking, Package, Routes, Enquiry, Feedback)
